# 🍳 Code-a-Cuisine

Never stare at your fridge without ideas again.

**Code-a-Cuisine** is a smart AI-powered web application that generates recipes based on the ingredients you already have at home. It helps you save time, reduce food waste, and discover new meal ideas effortlessly.

---

## 🚀 Features

* 🧠 AI-generated recipes based on available ingredients
* 🥕 Simple ingredient input
* ⚡ Fast and dynamic recipe generation
* ♻️ Reduces food waste
* 💡 Inspires creative cooking

---

## 🛠️ Tech Stack

* **Frontend:** Angular, TypeScript, SCSS  
* **Backend / Database:** Supabase (PostgreSQL, Auth, API)  
* **Automation / Workflows:** n8n  
* **AI Integration:** OpenAI API (via n8n workflows)  

---

## 🧩 Usage

1. Enter the ingredients you currently have
2. Trigger the recipe generation
3. Get personalized recipe suggestions instantly

---

## 🎯 Vision

Code-a-Cuisine aims to make cooking smarter, more sustainable, and more creative by turning everyday ingredients into delicious meals.
