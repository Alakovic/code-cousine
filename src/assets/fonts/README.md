# 🧠 Fonts (quick note)

## Inter → everything

* 400 → text
* 500 → slight emphasis
* 600 → buttons / important
* 700 → bold

## Poppins → headings

* 600 → normal titles
* 700 → main titles

## SCSS

body { font-family: 'Inter', sans-serif; }
h1, h2, h3 { font-family: 'Poppins', sans-serif; }

## Use only

* Inter: 400, 500, 600, 700
* Poppins: 600, 700
